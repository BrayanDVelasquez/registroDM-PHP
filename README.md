# RegistroDM-PHP

Registro de doctores y maestros, pidiendo al usuario ingresar sus datos personales y otros laborales.
Sin base de datos ni sesiones, solo es un proyecto estatico con PHP (POO).

Framework CSS -> Bootstrap v4.
