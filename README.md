# RegistroDM-PHP

Registro de maestros y profesores, pidiendo al usuario ingresar sus datos personales y otros laborales.

Sin base de datos ni sesiones, solo es un proyecto estatico con PHP.
Framework CSS -> Bootstrap v4.
